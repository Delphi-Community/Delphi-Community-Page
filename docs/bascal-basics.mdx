# Pascal Basics

Welcome to the fundamentals of Pascal! This guide covers core syntax and concepts shared across **Free Pascal**, **Delphi**, and **Lazarus**.

## Hello World
```pascal
program HelloWorld;
begin
  WriteLn('Hello, Pascal!');
end.
```

## Variables & Data Types

### Core Data Types

These fundamental data types are common across all Pascal implementations:

```pascal
var
  // Integer types
  smallInt: ShortInt;    // 8-bit signed (-128..127)
  byteVal: Byte;         // 8-bit unsigned (0..255)
  intVal: SmallInt;      // 16-bit signed (-32768..32767)
  wordVal: Word;         // 16-bit unsigned (0..65535)
  longIntVal: LongInt;   // 32-bit signed (-2147483648..2147483647)
  dWordVal: DWord;       // 32-bit unsigned (0..4294967295)
  int64Val: Int64;       // 64-bit signed (very large range)
  qWordVal: QWord;       // 64-bit unsigned (0..18446744073709551615)
  integerVal: Integer;   // Platform-dependent signed integer
  cardinalVal: Cardinal; // Platform-dependent unsigned integer
  
  // Floating-point types
  singleVal: Single;     // 32-bit floating-point
  doubleVal: Double;     // 64-bit floating-point
  extendedVal: Extended; // 80-bit floating-point (where available)
  realVal: Real;         // Alias for Double in modern Pascal
  compVal: Comp;         // 8-byte integer for financial calculations
  currencyVal: Currency; // Fixed-point type for financial calculations

  // Character and string types
  charVal: Char;         // Single character (ASCII in classic Pascal)
  strVal: String;        // Dynamic string
  shortStrVal: ShortString; // String with max 255 characters
  
  // Boolean type
  boolVal: Boolean;      // True or False
  
  // Date and time
  dateVal: TDateTime;    // Date and time representation
```

### Type Ranges

| Type | Size (bytes) | Range |
|------|--------------|-------|
| ShortInt | 1 | -128..127 |
| Byte | 1 | 0..255 |
| SmallInt | 2 | -32,768..32,767 |
| Word | 2 | 0..65,535 |
| LongInt | 4 | -2,147,483,648..2,147,483,647 |
| DWord | 4 | 0..4,294,967,295 |
| Int64 | 8 | -9,223,372,036,854,775,808..9,223,372,036,854,775,807 |
| QWord | 8 | 0..18,446,744,073,709,551,615 |
| Single | 4 | 1.5E-45..3.4E38 (7-8 digits precision) |
| Double | 8 | 5.0E-324..1.7E308 (15-16 digits precision) |
| Extended | 10 | 3.4E-4932..1.1E4932 (19-20 digits precision) |
| Currency | 8 | -922,337,203,685,477.5808..922,337,203,685,477.5807 |

## Declaration Rules
- Declare variables in a var block.
- Initialize with :=:

```pascal
var
  count: Integer = 0;
```
