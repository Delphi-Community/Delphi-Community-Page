### UTF-8 Best Practices

1. **Be explicit about types** - Use explicit string types (`UTF8String`, `UnicodeString`) rather than relying on the default `string` type
   
2. **Be aware of length calculations** - `Length()` returns bytes for UTF-8 strings, not character count
   
3. **Source file encoding matters** - Use compiler directives to specify your source code encoding:
   ```pascal
   {$CODEPAGE UTF8}
   ```
   
4. **Use appropriate routines** - For character-aware operations on UTF-8 strings:
   - Delphi: Use `Character` unit functions
   - Free Pascal: Use `UnicodeUtils`, `LazUTF8` or `LConvEncoding` units
